**문제:** 새가 경계(x=800, x=3400)에 닿으면 `b.x = 11800`으로 텔레포트해서 화면에 순간 출현함.

**수정:** 텔레포트 대신 속도 부호 반전(`setVelocityX(±Math.abs(vx))`)으로 자연스러운 방향 전환.

### 📷 카메라

**문제:** 발사 순간 카메라가 새총 위치로 순간 이동(점프)하는 현상.

**원인:** `cam.startFollow()`가 호출 즉시 `setScroll()`을 실행해 돌의 드래그 위치로 카메라가 튐.

**수정:** `startFollow()` 제거 → `update()`에서 수동 scroll lerp(`Phaser.Math.Linear`, t=0.1)만 사용.

---

**문제:** 카메라 팔로우 + 줌 lerp 동시 실행 시 화면 끝에서 심한 프레임 드랍.

**수정:** 팔로우 중 줌 변경 완전 제거 — 수동 스크롤 lerp만 남겨 연산 분리.

---

**추가:** `wipeReturn()`에 nearHome 체크 — 카메라가 이미 홈 위치 근처(zoom≥0.95, 오차<200px)면 암전 트랜지션 생략하고 즉시 복귀.

### 🎯 조준 & 발사

**문제:** 궤적 예측선이 새총 뒤쪽에서 시작해 물리적으로 어색함.

**수정:** 궤적 시작점 및 실제 돌 발사 위치 모두 `SLING_CENTER`로 통일 (Angry Birds 스타일).

---

**문제:** 줌 아웃 상태에서 `pointer.worldX/Y`가 부정확해 조준 불가.

**수정:** `cam.scrollX + pointer.x / cam.zoom` 공식으로 직접 월드 좌표 계산.

---

**추가:** 줌 아웃 시 새총 조준을 막는 대신, 화면 정중앙에 반투명 **"조준화면으로"** 버튼 표시.
- 마우스 휠로 zoom < 1.0이 되면 자동으로 등장
- 클릭 시 zoom=1.0, 새총 중심으로 카메라 즉시 복귀
- 발사·wipeReturn·라운드 시작·게임 재시작 시 자동 숨김
- `compensateHUDZoom()`에서 `setScale(1/z)`로 어느 줌에서도 동일한 크기 유지

### 💥 명중 판정

**문제:** 새나 돌이 겹쳐도 판정이 안 됨 — hitbox가 실질적으로 ~12px 수준.

**원인:** `bird.setScale(0.5)` 후 `body.setSize(25, 25)` → Phaser 내부에서 `25 × _sx(0.5) = 12.5px`. 돌도 동일 문제.

**수정:** `setSize()` 호출 완전 제거 — 텍스처 원본 크기(50×50) 그대로 사용, 시각 크기(25px)보다 2배 큰 여유 hitbox 확보.

### 🗺️ 미니맵

**문제:** 미니맵에서 새들이 전부 왼쪽에 몰려 보임.

**원인:** `toMX = wx / WORLD_W` 공식이 WORLD_W=12000 기준이라 활동 구역(x=800~3400)이 전체의 28%에 불과.

**수정:** `MM_WX0=0, MM_WX1=4000` 상수 추가, `toMX = (wx - MM_WX0) / (MM_WX1 - MM_WX0) * ww`로 활동 구역이 미니맵 전체를 채우도록 보정.
