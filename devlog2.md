### ☑️ 패시브 토글 시스템

**추가:** 매 발사 전 패시브를 취사선택할 수 있는 토글 버튼 UI 구현.

- 보유 패시브가 생기면 화면 **하단 좌측부터** 차례로 버튼 등장
- `☑ 팔근육` (초록) / `☐ 농구공` (회색) — 클릭으로 ON/OFF 전환
- `passiveActive = {}` 전역 객체로 상태 관리, `isOn(name)` 헬퍼로 조회
- `rebuildToggleUI(scene)` — 패시브 획득·재시작 시 버튼 전체 재생성
- `permanentHitAreas`에 `isToggle: true` 마킹 → `clearOverlay()` 영향 없음
- `startDrag()` 에 토글 영역 클릭 차단 추가 (맵드래그 방지)
- `compensateHUDZoom()` 에 토글 버튼 줌 보정 추가

**토글 적용 범위:**

| 패시브 | OFF 시 |
|-------|--------|
| 팔근육 | `BASE_LAUNCH` 그대로 |
| 큰돌 | 기본 크기 0.5배 |
| 철구슬 | 데미지 1 |
| 조준선 | 궤적 미표시 |
| 농구공 | 바운스 없음 |
| 관통 | 맞으면 돌 소멸 |

### 🪃 부메랑 패시브 — 추가 후 삭제

**추가 시도:** 발사 후 자력으로 새총에 귀환, 관통과 조합 시 왕복 타격.

**문제점:**
1. `allowGravity = false` → 중력 없이 자력만 작용 → 이상한 위쪽 궤적
2. 시간 기반 트리거(1200ms) → 2초 후 힘없이 뚝 떨어지는 현상
3. 속도 기반 트리거로 교체했으나 궤적이 여전히 부자연스러움

**결론:** 게임성 대비 구현 난이도가 높아 **패시브 전체 삭제**.

**제거 범위:** `boomerangStack` 전역변수, update 루프 전체 블록, hitBird 분기, ALL_PASSIVES 항목, DEV MODE 항목, applyPassive/restartGame/refreshPlayerPanel 모두.
