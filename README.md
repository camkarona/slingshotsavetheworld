# THE SLINGSHOT TO RESCUE WORLD
### 세상을 구하는 새총 — Prototype v0.1

> 새총으로 오염 생물을 물리치는 싱글 플레이 로그라이크 게임

---

## 플레이 방법

1. `phase2a.html` 파일을 브라우저로 열기
2. 마우스로 돌멩이를 당겨서 발사
3. 50마리의 오염 생물을 전멸시키면 라운드 클리어
4. 돌멩이가 먼저 소진되면 게임오버

## 강화 시스템

- 박사님 연구 버튼 (우상단) → 특수물질 10g 소비 → 패시브 카드 3장 중 선택
- **NORMAL**: 탄성증가 / 파워업 / 돌을 줍다
- **RARE**: 큰돌 / 조준선 / 농구공 / 자석탄
- **EPIC**: 관통 / 산탄 / 폭발 / 번개탄

## 기술 스택

- [Phaser 3.60.0](https://phaser.io/) — 게임 엔진
- 순수 HTML + JavaScript (단일 파일)
- BGM 포함
